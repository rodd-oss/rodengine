# Repository Structure

## Root Directory
├── .husky/                    # Git hooks
│   └── pre-commit             # Pre-commit hook runs lint-staged
├── .gitignore                 # Git ignore patterns
├── .prettierignore            # Prettier ignore patterns
├── .prettierrc                # Prettier config (empty = defaults)
├── AGENTS.md                  # Instructions for AI agents (this file)
├── bun.lock                   # Bun lockfile
├── Cargo.toml                 # Rust workspace config
├── eslint.config.mts          # ESLint configuration
├── go.work                    # Go workspace config
├── index.ts                   # Main TypeScript file (currently empty)
├── package.json               # NPM/Bun package config with workspaces
├── README.md                  # Project documentation
├── tsconfig.json              # TypeScript config (strict mode)
└── REPO_STRUCTURE.txt         # This file

## Workspace Directories
apps/                          # Applications (currently empty except .gitkeep)
packages/                      # Shared packages (currently empty except .gitkeep)

## Documentation
docs/
└── new_requirments/
    ├── db_requirement.txt     # Database requirements
    └── db_trd.md              # Database technical requirements document

## Configuration Notes
- **Runtime**: Bun package manager/runtime
- **Workspaces**: apps/* and packages/* (monorepo)
- **Languages**: TypeScript (primary), Rust, Go workspaces defined
- **Linting**: ESLint with TypeScript, Vue, JSON, Markdown, CSS plugins
- **Formatting**: Prettier with default settings
- **Git Hooks**: Husky pre-commit runs lint-staged
- **TypeScript**: Strict mode with noUncheckedIndexedAccess enabled

## Key Commands
- `bun test` - Run tests
- `bunx eslint .` - Lint code
- `bunx tsc --noEmit` - Type check
- `cargo test` - Rust tests
- `go test ./...` - Go tests
- `prettier --list-different .` - Check formatting
